# AfroPerm Professional Tool

## Описание проекта
Данное программное обеспечение представляет собой интерактивный калькулятор, предназначенный для автоматизации расчета стоимости услуг студии AfroPerm. Приложение позволяет производить точный расчет стоимости плетения и коррекции дредов с учетом множества параметров: длины дред, количества изделий и дополнительных технологических операций.

## Технологический стек
Проект реализован как Progressive Web App (PWA) с использованием следующих технологий:
* **Frontend:** HTML5, CSS3 (фреймворк Tailwind CSS), JavaScript (ES6+).
* **Графический интерфейс:** Canvas API для визуальных эффектов.
* **Интеграция:** VK Bridge SDK для корректной работы внутри инфраструктуры экосистемы VK.
* **PWA:** Service Worker и Web App Manifest для обеспечения возможности автономной работы и установки на мобильные устройства.

## Функциональные возможности
1. **Калькуляция услуг плетения:** динамический расчет цены в зависимости от целевой длины (от 10 до 100 см) и общего количества изделий.
2. **Калькуляция услуг коррекции:** расчет базовой стоимости и дополнительных манипуляций (коррекция длины, прикорневое валяние).
3. **Модуль дополнительных услуг:** учет специфических операций, таких как оформление тупого кончика или сращивание (удлинение).
4. **Информационный модуль:** встроенный прайс-лист и технический регламент по распределению количества дредов в зависимости от густоты волос и зон головы (андеркат, челка).

## Структура репозитория
* `index.html` — основной исполняемый файл приложения.
* `manifest.json` — конфигурационный файл манифеста веб-приложения.
* `sw.js` — скрипт Service Worker для управления кэшированием и обеспечения офлайн-доступа.
* `icon.svg` — векторный графический ресурс логотипа.

## Развертывание и установка
Приложение не требует серверной компиляции и может быть развернуто на любых платформах статического хостинга (GitHub Pages, Vercel, Netlify).

### Локальный запуск
1. Клонируйте репозиторий.
2. Откройте файл `index.html` в современном веб-браузере.

### Использование как PWA
При доступе через HTTPS браузер предложит установить приложение на рабочий стол или главный экран мобильного устройства.
